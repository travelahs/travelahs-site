# Travelahs website

This version is curated for desktop and mobile and uses the strongest current portfolio content.

## Publish
Upload the contents of this folder to the root of a GitHub repository and enable GitHub Pages from the main branch/root folder. Then connect `travelahs.com` through Namecheap DNS.

## Update portfolio
Edit `content.js`. Each portfolio block is either:
- `feature`: one larger location with a text panel
- `duo`: two complementary locations side by side
- `landscape`: one full-width horizontal image

Keep image and video files in `media/` and reference them with relative paths.


Instagram section: The footer-area profile embed pulls the public @travelahs profile and recent posts directly from Instagram. It requires the account to remain public and Instagram embedding to remain enabled.
